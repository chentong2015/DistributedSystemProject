Apache Dubbo是一款易用、高性能的WEB和RPC框架
同时为构建企业级微服务提供服务发现、流量治理、可观测、认证鉴权等能力、工具与最佳实践

Apache Dubbo特点:
1. 基于代理的远程调用能力，服务以接口为粒度
2. 负载均衡策略: 使用注册中心，服务自动注册与发现
3. 可视化的服务治理与运维: 监控
